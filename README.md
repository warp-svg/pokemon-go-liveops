# Pokémon GO LiveOps Strategy Lab

A Next.js 15 executive-style dashboard for modeling Pokémon GO content cadence, release timing, and LiveOps strategy using mock local datasets.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Recharts
